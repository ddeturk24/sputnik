Dashboard Settings

Google Calendar

No API key needed. Just paste the public iCal URL from your Google Calendar.

To find it: Google Calendar → Settings → your calendar → "Integrate calendar" → Public URL to this calendar (the one ending in .ics)

Weather Location

Defaults to Omaha, NE if left blank. Uses Open-Meteo (free, no API key).

Find coordinates: search your address on Google Maps, right-click → the lat/lng numbers are right there.
